# OAuth providers

Sign in with Google / GitHub / Microsoft / Apple.

## When this fits

- B2B (Google Workspace, Microsoft 365 alignment)
- Developer audience (GitHub OAuth is expected)
- Consumer at scale (everyone has a Google account)

## When this doesn't fit

- Audience that distrusts third-party identity
- Compliance regimes that require direct identity verification
