# Passkeys (WebAuthn)

The modern passwordless standard. Device-bound credentials, no shared secrets.

## When this fits

- Users on modern devices (iOS 16+, Android 9+, modern browsers)
- You want the strongest UX (Face ID / Touch ID login)
- Repeat-login is frequent

## When this doesn't fit

- Audience using older devices
- Cross-device flows that aren't supported by the user's password manager
- You can't fall back gracefully
